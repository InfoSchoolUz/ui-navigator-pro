🚀 UI Navigator Pro AI

Gemini AI Desktop Automation UI Understanding

What it does

UI Navigator Pro AI captures user interfaces, analyzes them using Gemini AI, and generates step-by-step navigation instructions.

How it works

  • Capture screen
  • Send to Gemini AI
  • Generate actions
  • Execute navigation
