🚀 UI Navigator Pro AI

An AI-powered assistant that understands user interfaces, analyzes screen content with Gemini AI, and generates step-by-step navigation actions for desktop workflows.

Gemini AI Desktop Automation UI Understanding Google Cloud Ready

What it does

UI Navigator Pro AI captures user interfaces, analyzes them using Gemini AI, and produces intelligent step-by-step navigation instructions for the user.

The goal is to help users interact with software more easily by combining visual understanding, action planning, and automation support.

How it works

  • Capture the current screen or interface
  • Send the UI context to the backend
  • Analyze the interface using Gemini AI
  • Generate structured navigation actions
  • Return guidance for desktop execution

Backend (Gemini Integration)

The backend is designed to process UI data and call Gemini AI for intelligent interpretation and action generation.

from google import genai client = genai.Client(api_key="YOUR_API_KEY") response = client.models.generate_content( model="gemini-2.0-flash", contents="Analyze this interface and suggest the next user action." )

☁️ Google Cloud Integration

This project is structured for Google Cloud deployment. The backend is containerized with Docker and prepared for cloud hosting on Google Cloud Run.

  • ✅ Containerized backend using Docker
  • ✅ API service architecture ready for deployment
  • ✅ Gemini-based cloud processing workflow
  • 🔄 Google Cloud Run deployment in progress

Planned Cloud Architecture

  • Desktop Client → Cloud API
  • Cloud API → Gemini AI analysis
  • Gemini AI response → Action generation
  • Generated instructions → User guidance / execution
Target platform: Google Cloud Run
AI engine: Gemini API
